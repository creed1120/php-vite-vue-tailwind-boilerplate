# PHP| Vite | Vue3 | Tailwind Boilerplate

Welcome to your Vite-powered project!

## Getting Started

1. Install dependencies:
    ```bash
    npm install
    ```
2. Start the development server:
    ```bash
    npm run dev
    ```

## Project Structure

- `src/` - Source files
- `public/` - Static assets
